# Enderpearl Backport

Backports the improved Ender Pearl mechanics from Minecraft 1.21.2+  for older versions of the game.

This repository is the **main entry point** for the mod.
Source code is split by mod loader.

---

## Source Code

- **NeoForge version:** [`/neoforge`](./neoforge)
- **Fabric version:** [`/fabric`](./fabric)

---

## Issues & Bug Reports

Please create all issues **in this repository** and specify:

- Mod Loader: Fabric / NeoForge
- Minecraft version
- Mod version
- Logs
